# StockDividend

Expert Advisor para MetaTrader 5 focado em estrategia de dividendos em acoes brasileiras.

## Download

### Instalador Automatico (Recomendado)

1. Baixe o [StockInstaller.exe](https://github.com/brenoperucchi/StockDividend/releases/latest/download/StockInstaller.exe)
2. Execute o instalador
3. Clique em "Detectar MT5" para encontrar suas instalacoes
4. Clique em "Instalar" para copiar o EA automaticamente

O instalador verifica atualizacoes automaticamente.

### Instalacao Manual

1. Baixe [StockDividendEA.ex5](https://github.com/brenoperucchi/StockDividend/releases/latest/download/StockDividendEA.ex5)
2. Copie para `[MT5]/MQL5/Experts/`
3. Reinicie o MetaTrader 5
4. Anexe o EA ao grafico

## Requisitos

- MetaTrader 5 (corretora com acoes brasileiras)
- Windows 10/11
